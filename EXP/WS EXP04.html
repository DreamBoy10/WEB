ir 효과

01. 의미 있는 이미지의 대체 텍스트를 제공하고자 하는 경우
.ir_pm {
display: block;
overflow: hidden;
font-size: 0;
line-height: 0;
text-indent: -9999px;
}
02. 의미 있는 이미지의 대체 텍스트로 이미지가 없어도 대체 텍스트를 보여주고자 하는 경우
.ir_wa {
display: block;
overflow: hidden;
position: relative;
z-index: -1;
width: 100%;
height: 100%;
}
03. 대체 텍스트가 아닌 접근성을 위한 숨김 텍스트를 제공하고자 하는 경우
.ir_su {
overflow: hidden;
position: absolute;
width: 0;
height: 0;
line-height: 0;
text-indent: -9999px;
}
